What are the differences between local storage vs session storage?

Local Storage:
1. Local storage don't creats with every single new tab specifically. 2. Local storage collects data till no one delete or clear them. 3. Local storage size is 5mb to 10mb


Session Storage:
1. Session storage creates with every new tab creation. 2. It holds data for the time user works on that tab. 3. It's data is cleared with closing of browser or perticular tab. 4. Session storage size is 5mb.

What are the differences between global scope vs block scope?

Global Scope:
1. The variables defined outside of any function or curly brackets are known as global variables and have global scope. Global scope means that the variables can be accessed from any part of that program, any function or conditional state can access that variable.
